# dlstar

**Fast stream file telegram on direct link**   

install:  
    - add config keys in */webgram/config.py  
    - install docker & docker-compose  
    - `docker-compose up`  

Heroku:  
    - add config keys in */webgram/config.py  
    - install Heroku Cli  
    - `heroku login`  
    - `heroku create` or `git:clone -a app`  
    - `heroku stack:set container`  
    - `python deploy.py`
